本仓库提供一套通用的 iOS App 开发与上架流程技能（面向中国区），用于帮助开发者快速梳理从开发、构建、签名、测试到 App Store Connect 提交与审核的全流程。技能内容包含上架前的关键清单、常见审核注意事项、隐私合规要求、ATS 配置建议，以及订阅/IAP 的创建与测试步骤，适合独立开发者或小团队在发布前复用与检查。

包含内容：

SKILL.md：技能触发与输出结构（输入 iosdev 即可调用）
references/：按主题拆分的参考资料
构建/签名/依赖与版本号管理
上架流程与审核要点（中国区）
订阅（IAP）配置与测试
隐私、ATS 与安全合规
使用方法：

将本目录复制到 ~/.codex/skills/helloagents/
在 Codex 中输入 iosdev，即可输出标准流程与注意事项
需要细节时，可直接查看 references/ 中对应文件
