# 隐私、安全与合规要点

## 1. 隐私政策
- 必须提供可公开访问的网页链接（非 PDF）
- 明确说明：收集的数据类型、用途、共享对象、保留期限、用户权利

## 2. App Privacy（App Store Connect）
- 如实填写数据收集项与用途
- 若不收集数据，明确勾选“未收集”
- 涉及追踪/广告需评估 ATT 要求

## 3. 权限说明
- 麦克风/相机/定位/通知等权限需配置用途描述
- 说明内容需要与实际功能一致

## 4. ATS 与网络安全
- 优先使用 HTTPS
- 避免开启 `NSAllowsArbitraryLoads`
- 如必须使用 HTTP，配置域名级别例外

## 5. 密钥与敏感信息保护
- 禁止在客户端明文写入 API 密钥
- 使用服务端代理或环境注入
- 日志中避免输出用户隐私与密钥

## 6. 中国区合规提示（可能适用）
- 如涉及个人信息处理，需遵守《个人信息保护法》等要求
- 若涉及内容发布/UGC/地图/金融/医疗等，可能需要额外资质
- 以上以最新法规与审核政策为准
