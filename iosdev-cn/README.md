# iOSdev（iosdev-cn）

通用 iOS App 开发、构建、签名、测试与 App Store 上架流程技能（面向中国区）。

## 适用场景
- iOS 开发与发布流程梳理
- TestFlight / App Store Connect 上架与审核准备
- 订阅（IAP）配置与测试
- 隐私合规、ATS 与权限说明

## 目录结构
- `SKILL.md`：技能触发与工作流程
- `references/`：分主题参考资料
- `iosdev-cn.skill`：可分发的技能包

## 使用方式
1. 将本目录复制到 `~/.codex/skills/helloagents/`
2. 或使用 `iosdev-cn.skill` 进行分发安装
3. 在 Codex 中输入触发词 `iosdev` 即可调用
